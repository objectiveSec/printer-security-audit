# Checklist

## Первые действия

- [ ] Остановить активную печать
- [ ] Сохранить Job Log / Event Log
- [ ] Проверить, повторяются ли `Guest` и `source_IP=0.0.0.0`
- [ ] Проверить, есть ли одинаковые задания подряд
- [ ] Проверить, не печатается ли мусор/PCL/PostScript-текст

## Проверка сети

- [ ] Узнать IP принтера
- [ ] Проверить, доступен ли Port 9100
- [ ] Проверить, печатают ли ПК напрямую на IP принтера
- [ ] Проверить, есть ли print server
- [ ] Проверить ACL/firewall для подсети принтеров

## Проверка рабочих станций

- [ ] Проверить очереди печати
- [ ] Перезапустить Print Spooler на подозрительных ПК
- [ ] Удалить зависшие задания
- [ ] Проверить драйвер
- [ ] Удалить прямые IP-подключения к принтеру, если они не нужны

## Hardening

- [ ] Отключить RAW/Port 9100, если он не используется
- [ ] Разрешить печать только с print server
- [ ] Ограничить доступ к web-интерфейсу принтера
- [ ] Сменить пароль администратора
- [ ] Отключить лишние протоколы
- [ ] Обновить прошивку принтера
